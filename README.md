# WEB_projesi
## proje açıklaması
Bu proje, öğrencilerin akademik hayatlarını daha planlı, verimli ve motive bir şekilde sürdürebilmeleri amacıyla geliştirilmiş etkileşimli bir web uygulamasıdır. görev yönetimi, hedef belirleme ve motivasyon desteği sunarak öğrencilerin zaman yönetimini geliştirmelerine yardımcı olur.
## hedef kullanıcılar 
Üniversite öğrencileri
Lise öğrencileri
Sınava hazırlanan öğrenciler
Günlük görevlerini planlamak isteyen kullanıcılar
## temel özellikler 
kullanıcı kayıt/giriş 
görev ekleme 
görev silme
görev güncelleme
Görevleri tamamlandı olarak işaretleme
Günün motivasyon sözü
Hedef belirleme alanı
Verimli ders çalışma önerileri
Zaman yönetimi rehberi
